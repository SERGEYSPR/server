Тип метода: GET
Адрес (host): http://107.170.103.201/api/students.get
Параметры:
uids
Список идентификаторов студентов через запятую (Пример: 169,170,250). Этот параметр может быть пропущен, тогда сервер вернет весь список студентов
fields
Список названий полей через запятую. Этот параметр может быть пропущен, тогда сервер вернет полную информацию о студенте(ах)
Пример запроса:
http://107.170.103.201/api/students.get?uids=166,169,170&fields=first_name,birthday
Пример ответа на запрос выше:
[{"first_name":"Татьяна","birthday":"24.01.1993"},{"first_name":"Александр","birthday":"08.10.1992"},{"first_name":"Александр","birthday":"13.07.1994"}]
В случае ошибки ответ будет следующим:
{"error":{"message":"сообщение ошибки"}}
